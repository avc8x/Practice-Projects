# Practice-Projects

A collection of small practice projects built to strengthen core programming and object-oriented programming skills.

## Projects Included

### XO Game (C# .NET)
- Console-based XO game
- Focused on logic, loops, and condition handling

### Bank System (C++)
- Console-based bank management system
- Implemented using object-oriented programming principles
- Includes classes, encapsulation, and basic data handling

### Pizza Restaurant App (C# .NET)
- Console-based restaurant application
- Simulates ordering, menu handling, and basic business logic
- Focused on clean structure and user input handling

## Purpose
These projects were built for learning and practice purposes to improve problem-solving, OOP fundamentals, and console application development.
